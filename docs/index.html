Everything you need

A complete library
in your hands

From casual readers to serious bibliophiles — Bookshelf has every tool to manage, read, discover, and share your collection without giving away your data.

Multi-Format Reader

One built-in reader handles EPUB, PDF, and comic archives (CBR, CBZ). No extra apps needed — your whole collection reads in the browser.

Customizable Reading

Switch between Light, Dark, and Sepia themes. Choose your font (Sans, Serif, or Mono) and adjust size. Your preferences persist across sessions.

Text-to-Speech

Let the app read to you. Integrated TTS supports English, Italian, French, and Spanish voices — hands-free reading while you cook, commute, or rest.

Library Organization

Browse by authors, genres, publishers, or your own custom reading lists. Full search, filters, and sorting across your entire collection.

Reading Progress

Visual progress bars track how far you've come. Bookshelf remembers exactly where you left off and picks up right where you stopped — every time.

Reviews & Ratings

Write reviews and rate books 1 to 5 stars. Build a personal record of what you've loved, want to revisit, or would recommend to others.

Send to Kindle

Love your Kindle? Send any book directly to your Kindle email address with one click. Configure once and enjoy seamless cross-device delivery.

OPDS Catalog Feed

Bookshelf exposes a full OPDS 1.2 catalog. Connect KyBook 3, Thorium Reader, Moon+ Reader, Aldiko, or any compatible app to your library.

PWA — Install Anywhere

Add Bookshelf to your home screen on any device. It behaves like a native app — full-screen, snappy, always available at a tap.

Self-Hosted & Private

Your books live on your server. No vendor lock-in, no tracking, no fees. Deploy on a VPS, Raspberry Pi, or your home server in minutes.

Getting started

Up and running
in minutes

Bookshelf is built to be simple to deploy. Node.js backend + React frontend means standard tools just work — no exotic dependencies, no containers required.

01
Clone & Install

Clone the repository, run npm install in both frontend and backend directories. SQLite initializes automatically on first run — no database setup needed.

~3 minutes
02
Configure & Deploy

Set your server URL and optional email settings in the config file. Run with PM2 behind Nginx for production, or just start the Node server locally. Full deployment guide included.

Ubuntu · Nginx · PM2
03
Add Your Books

Upload books via the browser, point Bookshelf at a folder on your server to scan, or use the bulk upload tool. Add covers, metadata, and start reading immediately.

EPUB · PDF · Comics
# Clone and install
$ git clone https://github.com/vivamau/bookshelf.git && cd bookshelf
$ npm install --prefix backend && npm install --prefix frontend

# Build frontend and start
$ npm run build --prefix frontend
$ npm start --prefix backend

✓ SQLite database initialized
✓ Default admin user created
✓ Bookshelf running at http://localhost:3000
$

Multi-user support

Share your library
the right way

Bookshelf supports multiple users with a fine-grained permission system. Run it just for yourself, or open it to friends and family — you control who can do what.

Librarian
The Curator

Full control over the library and its users. Add books, manage metadata, scan directories, and administer the entire server.

  • Upload and manage books
  • Scan server directories
  • Manage users & roles
  • Read and download all content
Reader
The Bookworm

Everything needed for a great reading experience. Browse the full library, read, download, review, and build personal reading lists.

  • Read all books in the library
  • Download books
  • Write reviews & rate
  • Create custom reading lists
Guest
The Visitor

Browse and discover without full access. Guests can explore the catalog and see what the library contains before requesting a role.

  • View library catalog
  • Browse authors & genres
  • No download access
  • No reading access
